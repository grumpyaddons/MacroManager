# MacroManager addon
Tested on Classic WOTLK but should work on retail.

Type `/mm` or `/macromanager` to open the UI.

Built by `Grumpy` - `Grumpyd` on Nightslayer (Anniversary), `Grumpy` on Nazgrim (MoP Classic)

### Features
- Clean and streamlined UI.
- Search and filter your macros.
- Uses the built-in macro system and your existing macros.
- Easily move macros between account-wide and character-specific.
- Share macros in chat like you share WeakAuras.
- Browse read-only snapshots of your other characters' macros, captured the last time each of them logged in.
